# Mobile portal redesign — Amazon A to Z inspired

Planned for employee portal (/portal) mobile view. Build after web app is complete.

## Changes from current

### Header
- Notification bell with unread dot (top right)
- User avatar/initials (top right)
- Today's shift block front and center — time, role, location
- Clock in + Swap shift buttons inline on the shift block

### Home screen sections
1. Stats row — hours this week, PTO remaining
2. Notifications — swap approvals, PTO status, callouts (in-app, not just email)
3. Onboarding tasks — checklist with progress bar, disappears when 100% done
4. Documents preview — shows what's on file vs. needed

### Nav
Adds a 5th tab: Docs (alongside Home, Schedule, Time, Messages)

## Features needed before mobile redesign
